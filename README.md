<h1 align="center">KBBI-Offline</h1>

<p align="center">
Aplikasi Kamus Besar Bahasa Indonesia (KBBI) Luring untuk Windows.
</p>


## Instalasi
1. Unduh arsip .zip versi terbaru dari halaman **[Releases](../../releases)**
2. Ekstrak file .zip tersebut  
3. Jalankan aplikasi dengan membuka file `kbbi.exe`
